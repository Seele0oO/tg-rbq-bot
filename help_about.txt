===== 关于本 BOT ====
本 bot 为绒布球相关指定群组提供服务。
功能只在指定的群组中生效，不要将其添加到其他群组中，将会拒绝服务。
注意：要使用本 bot ，你的 Telegram 账户必须已经设置 username 用户名（以 @ 开头，不是 display name 显示名称）。
===== 隐私权 =====
本 bot 不会主动收集聊天内容。但本 bot 所回复和被回复的消息、以及程序出错时的报错调试信息可能会被后台命令行输出，以用作调试等情况，因此遭到命令行临时性地记录，但不会永久保存在硬盘中。
本 bot 不会与任何第三方 API 取得通信。
本 bot 具有管理员权限，仅为实现玩法和功能，不会进行滥权。如果有异常管理员操作（非以上描述中之功能）可作为 BUG 反馈。
由 KagurazakaYashi 创建的实例将不会使用任何位于国内机房或公司位于中国内地的云服务运营商。自建实例的话也强烈建议如此。
不要在 @ 本 bot 或带有本 bot 指令的情况下，附带个人的隐私信息。
程序作者和实例部署者不为因使用本软件而造成的后果承担任何责任。
===== 许可和服务 =====
许可：在使用本程序或源代码时请遵守 MIT License 并且禁止违反当地法律之用途。
不保证服务质量：本 bot 可能会随时进入维护甚至生产环境在线开发并不另行通知，如果突然停止响应请稍后再试，因为请求可能会产生堆积而在恢复服务后同时执行而发生预期外的行为。
滥用警告：对本 bot 的滥用行为可能会被管理员加入黑名单而被禁止使用本 bot 中的任何命令。本 bot 具有积分功能，但严禁为了刷分而在群组中刷屏，这与增加群聊乐趣的初衷本末倒置。
